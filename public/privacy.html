Privacy Policy

VORTEX: Cosmic Arena

Last updated: March 4, 2026

Information We Collect

VORTEX: Cosmic Arena is a real-time multiplayer browser game. We collect minimal data necessary to provide the game experience:

  • Gameplay Data: Player names (chosen by users), game scores, and in-game statistics for the duration of a game session.
  • Technical Data: Basic connection information required for real-time multiplayer functionality (WebSocket connections).
  • Purchase Data: If you make in-app purchases, payment processing is handled entirely by Stripe. We do not store credit card numbers or payment details on our servers.

How We Use Information

We use collected information solely to:

  • Provide and maintain the multiplayer game experience
  • Display leaderboards during gameplay
  • Process in-game coin purchases through Stripe

Data Storage and Retention

Game session data is temporary and stored in memory only during active gameplay. We do not maintain persistent user accounts or long-term data storage of gameplay information. When a game session ends, session data is cleared.

Third-Party Services

We use the following third-party services:

  • Stripe: For processing payments. Stripe's privacy policy can be found at stripe.com/privacy.

Children's Privacy

VORTEX: Cosmic Arena is not directed at children under 13. We do not knowingly collect personal information from children under 13.

Data Security

We implement appropriate security measures to protect the limited data we process. All connections use HTTPS/WSS encryption.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated date.
